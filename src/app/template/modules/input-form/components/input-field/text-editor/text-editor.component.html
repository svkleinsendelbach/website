Für einen langen Text, wählen Sie bitte 'Blocksatz' aus.

Um einen Link hinzuzufügen, gehen Sie wie folgt for:

1. Wählen Sie die gewünschte Schriftgröße aus.

2. Fügen Sie den Link ein. Dieser Link darf nicht mit 'http' oder 'https' beginnen.

3. Ändern Sie die Farbe des Links zu 'Highlight Farbe'.
